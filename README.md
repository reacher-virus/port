# Yash Agnihotri — Portfolio

A responsive static portfolio for a backend and AI/ML engineer.

## Run it locally

Open `index.html` directly in a browser, or serve this folder with any static-file server.

## Keep these details up to date

- **CV:** replace `yash.pdf` in this folder. The Download CV button will keep working as long as the filename stays the same.
- **Projects:** update each project link and copy in `index.html` as repositories or live demos become available.
- **Contact:** your email and social links are near the bottom of `index.html`.
- **Styles:** all layout, colours and responsive rules live in `styles.css`.

## Deploy

This site has no build step, so it can be deployed directly to GitHub Pages, Netlify, Vercel or any static host.
